# ASUS ROG Store

Trang web bán sản phẩm ASUS ROG với đầy đủ chức năng.

## Tính năng
- Trang chủ với sản phẩm nổi bật
- Trang sản phẩm theo danh mục
- Trang chi tiết sản phẩm
- Giỏ hàng
- Quản lý admin
- Quản lý kho hàng
- Đăng ký/đăng nhập

## Công nghệ
- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage

## Triển khai
Website được deploy trên Vercel: [https://vercel.com]([https://asus-rog-store.vercel.app](https://vercel.com/new?teamSlug=phongles-projects-396a4ecf))

## Tài khoản mẫu
- Admin: `admin@rogstore.vn` / `admin123`
- User: Đăng ký tài khoản mới

## Cài đặt local
1. Clone repository
2. Mở file `index.html` trong trình duyệt
3. Hoặc dùng Live Server (VS Code extension)
